How to set up:
  1. Create a new Google Sheet.
  2. Go to Extensions > Apps Script.
  3. Delete the code there and paste the code from gsheets.js (or the project help file).
  4. Click Deploy > New Deployment.
  5. Select Web App. Change access to "Anyone".
  6. Copy the Web App URL and paste it below.
Note: Each user on a different device needs their own URL for their own sheet.

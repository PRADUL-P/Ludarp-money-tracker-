## 🧠 Overview

LUDARP Money Tracker is a powerful, privacy-first personal finance web application designed to help users manage income, track expenses, and analyze financial data efficiently.

Built as a Progressive Web App (PWA), it works like a native app while running in the browser — fast, lightweight, and fully functional even offline.



---

## 🛡️ Core Philosophy

> Your data belongs to you — not to servers.

- 100% local storage by default  
- No login or tracking required  
- Works completely offline  
- Optional Google Sheets cloud sync  
- Encrypted manual backup support

## 🧭 App Structure

- ➕ Entry – Add and manage transactions  
- 📊 Summary – Analytics and reports  
- 🤝 Dues – Track money you owe or are owed  
- 🏦 Accounts – Manage balances  
- ⚙️ Settings – Full customization  

---

## ⚡ Entry Page (Command Center)

Designed for speed — log transactions in seconds.

### Features

- Expense / Income / Transfer modes  
- Smart category selection with custom categories  
- Voice input for quick entry  
- Built-in calculator for quick calculations  
- Date selection with backdate support  

### Advanced Features

- Fuel Tracker (Mileage, Cost per KM, Price tracking)  
- Travel expense tracking for future dates  
- Split expenses (Equal / Custom)  
- Auto-link to dues system  

---

## 📊 Summary Page (Analytics Hub)

Turn your financial data into insights.

### Features

- Monthly overview (Income, Expense, Net Flow)  
- Interactive pie charts  
- List and table view toggle  
- Daily and monthly grouped totals  
- Advanced filters (date, category, account)  

### Export Options

- Export to Excel (.xlsx)  
- Generate PDF reports  

---

## 🤝 Dues Page (Debt Manager)

Track all your pending and settled dues.

### Features

- Person-wise tracking  
- Advanced filters (amount, person, status)  
- Mark as Paid/Received with account tracking  
- Auto entry creation in transaction history  
- Undo settlement option  

---

## 🏦 Accounts & Net Worth

Manage all your financial sources in one place.

### Features

- Multiple accounts (Bank, Cash, Cards)  
- Opening balance setup  
- Auto-sync with transactions  
- Real-time balance updates  

---

## ⚙️ Settings (Control Tower)

Customize the app based on your needs.

### Features

- Profile customization  
- Currency selection (₹, $, €, etc.)  
- Category management with icons  
- Prefills (quick transaction shortcuts)  
- Drag-and-drop UI ordering  

### Security

- Biometric lock (Fingerprint / Face ID)  
- PIN protection  

### Backup & Sync

- Google Sheets sync (optional)  
- Encrypted JSON backup

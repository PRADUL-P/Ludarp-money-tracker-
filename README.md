## 🚀 Key Features

### 💎 Smart Dashboard
-   Real-time expense trends and category pie charts.
-   Filterable history logs with instant search capabilities.
-   Visual budget indicators (Automatic color-coding).

### 📥 Quick Entry & Voice Support
-   **Voice Log**: Hands-free expense logging using speech-to-text.
-   **Prefills**: One-tap logging for repetitive expenses like Rent, Milk, or Commute.
-   **Split Bill**: Advanced group split (Equal or Custom) with auto-linking to the Dues Tracker.

### 🏧 Multi-Account Net Worth
-   Track Bank Balances, Credit Cards, and Wallets in one place.
-   Automatic account balance adjustments when transactions are logged.
-   Direct settlement mapping to specific bank accounts.

### 📋 Master Reporting
-   Professional tabular statements for any period.
-   Export filtered logs to **Excel (XLSX)** or **Professional PDF** formats.

## 🛠️ Technology Stack
*   **Core**: Pure HTML5 & Vanilla JavaScript (Zero bloated frameworks).
*   **Styling**: Modern CSS3 with CSS Variables and Glassmorphism.
*   **Offline**: Service Worker (PWA) for 100% offline usage and native-app feel.
*   **Storage**: Browser LocalStorage & IndexedDB (Privacy focused).
*   **Export**: SheetJS (Excel) & jsPDF (Reports).

## 📦 How to Install
Since LUDARP is a PWA, you can install it on any device:

1.  **Mobile (Android/iOS)**: Open the URL in Chrome/Safari -> Add to Home Screen.
2.  **Desktop**: Click the "Install" icon in the URL bar (Chrome/Edge).
